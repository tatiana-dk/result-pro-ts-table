# Универсальная таблица на TypeScript

Лёгкая, типобезопасная и расширяемая таблица с поддержкой поиска, сортировки, фильтров, пагинации и сохранения состояния в localStorage.

Проект создан как демонстрация / библиотека для быстрого создания таблиц с данными любой структуры.

## Демонстрационные данные

**Тема данных**: картины импрессионистов и постимпрессионистов (включая русских художников)

Примеры полей одной записи:

- `id` — уникальный идентификатор
- `title` — название картины
- `artist` — имя художника
- `year` — год создания
- `medium` — техника (масло на холсте, пастель и т.д.)
- `isPublicDomain` — находится ли в общественном достоянии (boolean)

Данные включают работы Клода Моне, Пьера-Огюста Ренуара, Игоря Грабаря, Константина Коровина, Винсента Ван Гога и других.

## Возможности (на текущий момент)

| Функция                 | Реализовано | Примечание                                                                 |
|-------------------------|-------------|----------------------------------------------------------------------------|
| Глобальный поиск        | ✓           | по всем текстовым полям, без учёта регистра, с debounce                   |
| Сортировка по столбцу   | ✓           | клик по заголовку → asc → desc → сброс, стабильная сортировка             |
| Фильтры по колонкам     | ✓           | текст (contains), число (диапазон), дата (до/после/между), boolean        |
| Пагинация               | ✓           | кнопки «Назад» / «Вперёд», выбор количества строк на странице             |
| Сохранение состояния    | ✓           | поиск + фильтры + сортировка + текущая страница + размер страницы в localStorage |
| Сброс всех настроек     | ✓           | кнопка «Сбросить всё» с подтверждением                                    |
| Типобезопасность        | ✓           | строгие типы колонок, состояний, значений ячеек                           |
| Иммутабельность         | ✓           | исходные данные не мутируются                                             |
